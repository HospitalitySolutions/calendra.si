# iOS app scaffold

This folder contains a native SwiftUI app scaffold intended to consume the KMP `SharedGuest` framework from `../shared`.

Because Xcode cannot be executed here, the project is provided as:
- `GuestMobile.xcodeproj`
- `project.yml` reference spec
- SwiftUI source files
- `build-kmp-framework.sh` helper script

Open the project in Xcode and adjust signing, team, and bundle identifiers as needed.
